    <h2>Lesson One: If Conditional</h2>
    <p><strong>Dialogue:</strong></p>
    <p><em>I look</em></p>
    <p><strong>Mohamed:</strong> What will you wear tomorrow?</p>
    <p><strong>Abdel Malik:</strong> I will wear a jacket because it's rainy and the weather is very cold ❄️🥶</p>

    <p><strong>The Rule:</strong></p>
    <p>There are four types of the conditional:</p>

    <h3>Type 0</h3>
    <p>It's used to explain scientific facts. Everything here will happen 100%.</p>
    <p><strong>Example:</strong> "If it is hot, ice melts."</p>
    <p>In this type, we use the present simple in both sentences.</p>

    <h3>Type 1</h3>
    <p>In this type, not everything will happen, we can say about 80% probability.</p>
    <p><strong>Example:</strong> "If you work hard, you will succeed."</p>
    <p>In this type, the first sentence uses present simple, and the second sentence uses future simple tense.</p>

    <h3>Type 2</h3>
    <p>In this type, what we talk about won't happen now, but it could happen in the future.</p>
    <p><strong>Example:</strong> "If I were you, I would open the store."</p>
    <p>Here, the first sentence uses past simple, and the second sentence uses modal verb "would" + base verb.</p>

    <h3>Type 3</h3>
    <p>In this type, what we talk about will never happen.</p>
    <p><strong>Example:</strong> "If this player played with us, we would win the match."</p>
    <p>The first sentence uses past simple, and the second uses modal verb "would" + base verb.</p>

    <h3>Unless</h3>
    <p>"Unless" works like "if" but means "if not".</p>
    <p><strong>Example:</strong></p>
    <p>"If you work hard, you won't fail."<br>
       "Unless you work hard, you will fail."</p>

    <p><strong>That's all in this lesson.</strong></p>
